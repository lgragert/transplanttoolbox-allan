transplanttoolbox_allan.conversion_functions.convert_allele_to_ag(allele)[source]¶

This function should be called when antigen conversion has to be done for a single allele. Enter an IMGT/HLA allele as a string input and the get the coressponding antigen as per UNOS rules. Also if the antigen is Bw4/6 is also indicated. The rule applied for the conversion is printed out with every input
